# Frontend - Student Feedback Tracker

React frontend for the Student Feedback Tracker application.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

The app will open at http://localhost:3000

## Features

- Submit feedback form with name and message
- Real-time feedback display
- Responsive design
- Form validation
- Beautiful UI with animations

## Technologies

- React 18
- Axios
- CSS3

## Available Scripts

- `npm start` - Start development server
- `npm build` - Build for production
- `npm test` - Run tests
